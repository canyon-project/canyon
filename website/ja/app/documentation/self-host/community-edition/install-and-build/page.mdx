# インストールとビルド

Canyonコミュニティ版のインストールとビルド方法について学びます。

## 環境の設定

インストールを開始する前に、環境変数を設定する必要があります。

```yaml
# Prisma Config
DATABASE_URL=postgres://username:password@localhost:5432/dbname
```

## Node.js

プロジェクトをクローン

```shell
git clone https://github.com/canyon-project/canyon
```

依存関係をインストールしてプロジェクトをビルド

```shell
pnpm install && pnpm run build
```

.envファイルを修正し、データベーステーブルを作成

```shell
pnpm run migrate
```

Node.jsでプロジェクトを実行

```shell
node packages/canyon-backend/dist/main.js
```

## Docker

Dockerを使用してcanyonを実行するには、環境変数ファイルのパスとポート番号を指定するだけです

```shell
docker run -d -p 8080:8080 -v /your/path/.env:/app/.env zhangtao25/canyon:main
```

## 起動成功

起動成功後、2つの主要なサービスがあります：

- Canyon frontend
- Canyon backend

`http://localhost:8000/login`にアクセスしてCanyonにログインします。
