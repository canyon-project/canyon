# インストールとビルド

Canyon コミュニティ版のインストールとビルド方法について説明します。

## 環境の設定

インストールを開始する前に、環境変数を設定する必要があります。

```yaml
# Prisma Config
DATABASE_URL=postgress://canyon:canyon@localhost:5432/canyon
```

## Node.js

プロジェクトのクローン

```shell
git clone https://github.com/canyon-project/canyon
```

依存関係のインストールとプロジェクトのビルド

```shell
pnpm install && pnpm run build
```

.envファイルの修正とデータベーステーブルの作成

```shell
pnpm run migrate
```

nodejsでプロジェクトを実行

```shell
node packages/canyon-backend/dist/main.js
```


## Docker

dockerでcanyonを実行する場合は、環境変数ファイルのパスとポート番号を指定するだけです

```shell
docker run -d -p 8080:8080 -v /your/path/.env:/app/.env zhangtao25/canyon:main
```

## 起動成功

起動が成功すると、2つの主要なサービスが利用可能になります：

- Canyon frontend
- Canyon backend

`http://localhost:8000/login` にアクセスしてCanyonにログインします。
