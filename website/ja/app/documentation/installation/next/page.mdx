# Next.js

import { Steps } from "nextra/components";

Next.jsのインストールと設定。

<Steps>

  ### プロジェクトの作成
  まず、以下のコマンドを使用して新しいReactプロジェクト`next.js`を作成します。

  ```sh npm2yarn copy
  npx create-next-app@latest
  ```

  ### Canyonとその設定の追加

  > [!TIP]
  >
  > 最新のnext.jsコンパイラーはデフォルトでswcを使用します。babelプラグインの設定については、[こちら](https://nextjs.org/docs/advanced-features/customizing-babel-config)をご覧ください。

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### babelプラグインの設定

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["next/babel"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### 完了
  [最初のカバレッジデータ - 確認](/documentation/getting-started/first-coverage#確認)を続けてご覧ください。
</Steps>
