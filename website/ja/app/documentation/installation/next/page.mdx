# Next.js

import { Steps } from "nextra/components";

Next.jsのインストールと設定。

<Steps>

  ### プロジェクトの作成
  まず、以下のコマンドを使用して新しいReactプロジェクト`next.js`を作成します。

  ```sh npm2yarn copy
  npx create-next-app@latest
  ```

  ### Canyonの追加と設定

  > [!TIP]
  >
  > 最新のNext.jsコンパイラはデフォルトでswcを使用します。Babelプラグインの設定については、[こちら](https://nextjs.org/docs/advanced-features/customizing-babel-config)をご覧ください。


  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### babelプラグインの設定

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["next/babel"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### 以上です
  [最初のカバレッジデータ - 確認](/documentation/getting-started/first-coverage#%E6%A3%80%E6%9F%A5)に進んでください。
</Steps>
