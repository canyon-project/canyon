# Canyon Extension

Canyon ExtensionはChromeの拡張機能で、手動テスト中のカバレッジデータ収集を支援します。

## インストール

[Chrome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof)からCanyon Extensionをダウンロードしてインストールしてください。

## 使用方法

1. プロジェクトページを開きます。
2. Canyon Extensionのポップアップをクリックします。
3. レポートオプションを設定します。
4. テストを開始します。
