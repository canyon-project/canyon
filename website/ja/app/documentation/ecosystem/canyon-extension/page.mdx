# Canyon Extension

Canyon ExtensionはChromeプラグインで、手動テスト過程でカバレッジデータの収集をサポートします。

## インストール

[Chrome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof)をクリックしてCanyon Extensionをダウンロード・インストールしてください。

## 使用方法

1. プロジェクトページを開きます。
2. Canyon Extensionのポップアップをクリックします。
3. 報告オプションを設定します。
4. テストを開始します。
