# Canyon アップローダー

Canyon アップローダーは、カバレッジデータを報告するためのバイナリツールです。Linux、macOS、Windowsの3つのプラットフォーム用のバイナリファイルを提供します。

最新バージョンは[GitHub Release](https://github.com/canyon-project/canyon/releases)からダウンロードできます。

## 機能

ローカルの`.canyon_output`ディレクトリ下のカバレッジデータを読み取り、集約して報告

## 使用方法

```bash
chmod +x ./canyon-uploader
canyon-uploader map --report-id=reportID
```
