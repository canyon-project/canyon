# Canyon 日本語ドキュメント

Canyon の日本語ドキュメントサイトです。

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## ビルド

```bash
# プロダクションビルド
npm run build

# プロダクションサーバーの起動
npm start
```

## 翻訳について

このサイトは中国語版（`website/cn`）を基に作成されています。新しいコンテンツを追加する際は、対応する中国語版のファイル構造を参考にしてください。

## 貢献

翻訳の改善や新しいコンテンツの追加については、[GitHub Issues](https://github.com/canyon-project/canyon/issues) でお知らせください。
