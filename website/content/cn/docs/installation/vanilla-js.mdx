# 原生 JavaScript

import { Steps } from "nextra/components";

安装和配置原生 JavaScript 项目。

<Steps>

  ### 创建项目

  创建一个简单的 HTML 项目结构：

  ```html filename="index.html" copy
  <!DOCTYPE html>
  <html>
  <head>
    <title>Canyon Demo</title>
  </head>
  <body>
    <div id="app"></div>
    <script src="./src/main.js"></script>
  </body>
  </html>
  ```

  ### 添加 Canyon 及其配置

  > [!TIP]
  >
  > 推荐使用 babel 方式，这是最简单和最稳定的配置方式。原生 JS 项目也可以使用 babel 进行代码转换。

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### 配置 babel 插件

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["@babel/preset-env"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### 配置构建工具（可选）

  如果您想要更好的开发体验，可以添加简单的构建工具：

  ```js filename="webpack.config.js" copy
  const path = require('path');

  module.exports = {
    entry: './src/main.js',
    output: {
      path: path.resolve(__dirname, 'dist'),
      filename: 'bundle.js',
    },
    module: {
      rules: [
        {
          test: /\.js$/,
          exclude: /node_modules/,
          use: {
            loader: 'babel-loader',
          },
        },
      ],
    },
  };
  ```

</Steps>
