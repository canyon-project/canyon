# Webpack

import { Steps } from "nextra/components";

Install and configure Webpack.

<Steps>

  ### Create Project

  Webpack [documentation](https://webpack.js.org/)

  ### Add Canyon and its configuration

  > [!TIP]
  >
  > We recommend using the babel approach, which is the simplest and most stable configuration method.

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### Configure babel plugin

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["@babel/preset-env"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### Configure webpack

  ```js filename="webpack.config.js" copy
  const path = require('path');

  module.exports = {
    entry: './src/index.js',
    output: {
      path: path.resolve(__dirname, 'dist'),
      filename: 'bundle.js',
    },
    module: {
      rules: [
        {
          test: /\.js$/,
          exclude: /node_modules/,
          use: {
            loader: 'babel-loader',
          },
        },
      ],
    },
  };
  ```

  ### That's it
  Continue to view [First Coverage Data - Check](/docs/getting-started/first-coverage#%E6%A3%80%E6%9F%A5).
</Steps>
