# Getting Started

How to install Canyon plugins in your application.

> [!NOTE]
>
> Currently mainly supports Babel plugins, we strongly recommend using the Babel approach, which is the simplest and most stable configuration method. SWC, Vite and other plugins are still in development. [Plugins](https://github.com/canyon-project/canyon/tree/main/plugins).
