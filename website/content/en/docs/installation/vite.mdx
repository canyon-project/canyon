# Vite

import { Steps } from "nextra/components";

Install and configure Vite.

<Steps>

  ### Create Project
  First, create a new React project `vite` using the following command.

  > [!TIP]
  >
  > Currently only supports Vite's React template

  ```sh npm2yarn copy
  npm create vite@latest
  ```

  ### Add Canyon and its Configuration

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### Configure Babel Plugin

  ``` ts filename="vite.config.ts" copy {7,8,9}
  import { defineConfig } from 'vite'
  import react from '@vitejs/plugin-react'

  // https://vite.dev/config/
  export default defineConfig({
  plugins: [react({
      babel:{
        plugins:["istanbul","canyon"]
      }
    })],
  })

  ```

  ### That's It
  Continue to [First Coverage Data - Verification](/docs/getting-started/first-coverage#verification).
</Steps>
