# Canyon Uploader

Canyon Uploader is a binary tool for reporting coverage data. It provides binary files for three platforms: Linux, macOS, and Windows.

The latest version can be downloaded from [GitHub Release](https://github.com/canyon-project/canyon/releases).

## Features

Read and aggregate coverage data from the local `.canyon_output` directory for reporting

## Usage

```bash
chmod +x ./canyon-uploader
canyon-uploader map --report-id=reportID
```
