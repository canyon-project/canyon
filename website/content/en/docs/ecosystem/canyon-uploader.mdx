# Canyon Uploader

Canyon Uploader is a binary tool for uploading coverage data. Provides binary files for linux, macOS, and windows platforms.

The latest version can be downloaded from [GitHub Release](https://github.com/canyon-project/canyon/releases).

## Features

Reads coverage data from the local `.canyon_output` directory and aggregates uploads

## Usage

```bash
chmod +x ./canyon-uploader
canyon-uploader map --report-id=reportID
```
