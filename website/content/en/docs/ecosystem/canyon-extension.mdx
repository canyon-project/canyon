# Canyon Extension

Canyon Extension is a Chrome plugin designed to help you collect coverage data during manual testing.

## Installation

Click [Chrome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof) to download and install Canyon Extension.

## Usage

1. Open your project page.
2. Click on the Canyon Extension popup.
3. Configure reporting options.
4. Start testing.
