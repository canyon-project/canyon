# Canyon Extension

Canyon Extensionは、手動テスト中にカバレッジデータを収集するのに役立つChromeプラグインです。

## インストール

[Chrome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof)をクリックしてCanyon Extensionをダウンロードしてインストールします。

## 使用方法

1. プロジェクトページを開きます。
2. Canyon Extensionのポップアップをクリックします。
3. レポートオプションを設定します。
4. テストを開始します。
