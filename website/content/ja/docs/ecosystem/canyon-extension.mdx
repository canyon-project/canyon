# Canyon Extension

Canyon Extensionは、手動テスト中にカバレッジデータを収集するのに役立つChrome拡張機能です。

## インストール

[Crome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof)をクリックしてCanyon Extensionをダウンロード・インストールしてください。

## 使用方法

1. プロジェクトページを開きます。
2. Canyon Extensionのポップアップをクリックします。
3. レポートオプションを設定します。
4. テストを開始します。
