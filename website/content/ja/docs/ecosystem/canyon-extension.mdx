# Canyon Extension

Canyon Extension 是一个Chrome插件，用于帮助您在手工测试过程中收集覆盖率数据。

## 安装

点击 [Chrome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof)下载安装 Canyon Extension。

## 使用

1. 打开您的项目页面。
2. 点击 Canyon Extension 的 popup。
3. 配置上报选项。
4. 开始测试。
