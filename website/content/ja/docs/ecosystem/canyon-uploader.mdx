# Canyon Uploader

Canyon Uploaderは、カバレッジデータをアップロードするためのバイナリツールです。linux、macOS、windowsの3つのプラットフォームのバイナリファイルを提供します。

最新バージョンは[GitHub Release](https://github.com/canyon-project/canyon/releases)からダウンロードできます。

## 機能

ローカルの`.canyon_output`ディレクトリからカバレッジデータを読み取り、集約してアップロードします

## 使用方法

```bash
chmod +x ./canyon-uploader
canyon-uploader map --report-id=reportID
```
