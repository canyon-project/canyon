# Lynx

import { Steps } from "nextra/components";

Lynxをインストールして設定します。

<Steps>

  ### プロジェクトを作成

  [Lynx](https://lynxjs.org/) ドキュメント

  ### Canyonとその設定を追加

  > [!TIP]
  >
  > Lynxコンパイラはデフォルトでswcを使用します。プラグイン設定関連[ドキュメント](https://lynxjs.org/zh/api/rspeedy/rspeedy.tools.swc.html#toolsswc-property)


  ```sh npm2yarn copy
  npm install swc-plugin-coverage-instrument swc-plugin-canyon -D
  ```

  ### swcプラグインを設定

  ```js filename="lynx.config.ts" {19,20} copy
  import { defineConfig } from '@lynx-js/rspeedy'
  import { pluginQRCode } from '@lynx-js/qrcode-rsbuild-plugin'
  import { pluginReactLynx } from '@lynx-js/react-rsbuild-plugin'

  export default defineConfig({
    plugins: [
      pluginQRCode({
        schema(url) {
          return `${url}?fullscreen=true`
        },
      }),
      pluginReactLynx(),
    ],
    tools:{
      swc:{
        jsc:{
          experimental:{
            plugins:[
                ['swc-plugin-coverage-instrument',{}],
                ['swc-plugin-canyon',{}]
            ]
          }
        }
      }
    }
  })

  ```

  ### これで完了
  [最初のカバレッジデータ - チェック](/docs/getting-started/first-coverage#%E6%A3%80%E6%9F%A5)を続けて確認してください。
</Steps>
