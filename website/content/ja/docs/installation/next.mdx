# Next.js

import { Steps } from "nextra/components";

安装和配置 Next.js 。

<Steps>

  ### 创建项目
  首先使用以下命令创建一个新的 React 项目 `next.js` 。

  ```sh npm2yarn copy
  npx create-next-app@latest
  ```

  ### 添加 Canyon 及其配置

  > [!TIP]
  >
  > 最新next.js编译器默认使用swc，有关babel插件的配置，请查看[这里](https://nextjs.org/docs/advanced-features/customizing-babel-config)。

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### 配置 babel 插件

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["next/babel"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### 就是这样
  继续查看 [第一个覆盖率数据 - 检查](/docs/getting-started/first-coverage#检查)。
</Steps>
