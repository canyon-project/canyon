# Webpack

import { Steps } from "nextra/components";

安装和配置 Webpack。

<Steps>

  ### 创建项目

  Webpack [文档](https://webpack.js.org/)

  ### 添加 Canyon 及其配置

  > [!TIP]
  >
  > 推荐使用 babel 方式，这是最简单和最稳定的配置方式。

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### 配置 babel 插件

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["@babel/preset-env"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### 配置 webpack

  ```js filename="webpack.config.js" copy
  const path = require('path');

  module.exports = {
    entry: './src/index.js',
    output: {
      path: path.resolve(__dirname, 'dist'),
      filename: 'bundle.js',
    },
    module: {
      rules: [
        {
          test: /\.js$/,
          exclude: /node_modules/,
          use: {
            loader: 'babel-loader',
          },
        },
      ],
    },
  };
  ```
