# Canyon Uploader

Canyon Uploader is a binary tool for reporting coverage data. It provides binary files for Linux, macOS, and Windows platforms.

The latest version can be downloaded from [GitHub Release](https://github.com/canyon-project/canyon/releases).

## Features

Aggregate and report coverage data from the local `.canyon_output` directory

## Usage

```bash
chmod +x ./canyon-uploader
canyon-uploader map --report-id=reportID
```
