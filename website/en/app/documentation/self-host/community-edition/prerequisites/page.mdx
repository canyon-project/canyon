# Prerequisites

Prerequisites for installing Canyon on your own infrastructure.

Canyon is a self-hosted API development platform packaged as a set of Docker containers. You can install and run Canyon on any operating system that can run [Docker Engine](https://docs.docker.com/engine). You can use Canyon on your local machine or on your chosen cloud provider.

## System Requirements

Canyon is designed to run well in both small and large deployments. The minimum requirements to run Canyon are an operating system that supports Docker and 4 CPU cores + 4GB RAM to generate build images, and at least 1 CPU core + 2GB RAM to host the generated output files.

### Install Node.js, npm, pnpm

Install [`Node.js`](https://nodejs.org/en) (v18+) and [`npm`](https://www.npmjs.com) (v9+).

- [Node.js + npm Installation Guide](https://nodejs.org/en/download)

Install [`pnpm`](https://pnpm.io) (v9 recommended).

- [pnpm Installation Guide](https://pnpm.io/installation)

Install [`Docker`](https://www.docker.com) (v20+).

- [Docker Installation Guide](https://docs.docker.com/engine/install)
