# 报告期组件说明

报告期组件是变更代码覆盖率报告中的核心 UI 组件，用于展示和筛选不同时间段的覆盖率数据。

[//]: # (![变更覆盖率报告界面]&#40;/change-coverage.png&#41;)

---

## 主要功能

### 1. 时间范围选择

* 支持选择不同的报告期（如：本周、本月、自定义时间范围）
* 可以快速切换查看不同时间段的变更覆盖率趋势

### 2. 覆盖率指标展示

* 显示当前报告期内的总体变更覆盖率
* 展示受影响语句数（Impacted Statements）和已覆盖语句数（Covered Statements）
* 提供覆盖率百分比和可视化图表

### 3. 数据筛选与对比

* 可以按文件、提交、分支等维度筛选数据
* 支持对比不同报告期的覆盖率变化趋势
* 帮助识别覆盖率下降或提升的时间节点

### 4. 交互功能

* 点击报告期可以展开查看详细数据
* 支持导出当前报告期的覆盖率报告
* 提供钻取功能，从汇总数据深入到文件级别和语句级别

---

## 使用场景

通过报告期组件，用户可以：

- **快速了解项目在特定时间段内的变更代码覆盖率情况**
- **识别覆盖率趋势，发现潜在的质量风险**
- **对比不同时期的覆盖率数据，评估测试改进效果**
- **为代码审查和发布决策提供数据支持**
