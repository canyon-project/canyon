## babel.config.js 项目快速配置

如果你的项目使用 `babel.config.js`（例如 React Native、部分 Babel 工程化项目），可以直接按下面最小配置接入 `@canyonjs/babel-plugin`。

### 1) 安装依赖

```sh npm2yarn copy
npm install -D babel-plugin-istanbul @canyonjs/babel-plugin
```

### 2) 配置 Babel

```js filename="babel.config.js" copy
module.exports = {
  presets: [
    // 按你的项目保留原有 preset
  ],
  plugins: ["istanbul", "@canyonjs/babel-plugin"],
};
```

完成后，继续查看[首次获取覆盖率数据 - 验证](/docs/getting-started/first-coverage#%E6%A3%80%E6%9F%A5)。
