# Getting Started

How to install Canyon plugins into your application.

> [!NOTE]
>
> Currently only Babel plugins are supported. SWC, Vite and other plugins are still in development. [Plugins](https://github.com/canyon-project/canyon/tree/main/plugins).

## Canyon Toolchain Compatibility

### Tooling

| Tool | Node.js Requirement |
|------|---------------------|
| `canyon-cli` | Node.js >= 16 |
| `babel-plugin-canyon` | Node.js >= 16 |
| `swc-plugin-canyon` | Node.js >= 16 |

### Instrumented Output

- **JavaScript syntax**: ES5
- **Runtime**: Browser / Node.js / React Native
