# Data Relations

PostgreSQL data model derived from `schema.prisma`. Grouped by domain, with Prisma foreign keys and logical associations called out separately.


> Solid lines: Prisma `@relation` foreign keys. Dashed lines: field-level conventions without Prisma relations.

```mermaid
    }

    Account {
        string id PK
        string accountId

        string userId FK
        string accessToken

        string identifier
        string value

        datetime createdAt
```

| Model | Table | Notes |
| --- | --- | --- |
| `User` | `canyon_next_user` | User; unique `email` |

| `Account` | `canyon_next_account` | OAuth / password; `onDelete: Cascade` |
| `Verification` | `canyon_next_verification` | Verification tokens; no FK |

| Model | Table | Notes |
| --- | --- | --- |
| `Repo` | `canyon_next_repo` | Repository metadata and config |

## Coverage

Core chain: `CoverageBuild` → `Coverage` (by scene) → `CoverageHit`. Map relations / maps / source maps are shared per build.

```mermaid
erDiagram
    CoverageBuild ||--o{ Coverage : "coverages"
    CoverageBuild ||--o{ CoverageMapRelation : "mapRelations"
    Coverage ||--o{ CoverageHit : "hits"
    Coverage ||--o| CoverageLock : "lock"
    CoverageMap ||--o{ CoverageMapRelation : "relations"
    CoverageSourceMap ||--o{ CoverageMapRelation : "relations"

    CoverageBuild {
        string buildHash PK
        string provider
        string repoID
        string sha
        string buildTarget
        string instrumentCwd
        json builds
        datetime createdAt
        datetime updatedAt
    }

    Coverage {
        string id PK
        string buildHash FK
        string sceneKey
        json scene
        datetime createdAt
        datetime updatedAt
    }

    CoverageHit {
        string id PK
        string buildHash FK
        string sceneKey FK
        string rawFilePath
        json s
        json f
        json b
        int inputSourceMap
        datetime createdAt
    }

    CoverageLock {
        string coverageID PK,FK
        datetime lockedAt
        string lockedBy
    }

    CoverageMapRelation {
        string id PK
        string fullFilePath
        string restoreFullFilePath
        string buildHash FK
        string coverageMapHash
        string fileContentHash
        string coverageMapKey FK
        string sourceMapHash FK
    }

    CoverageMap {
        string hash PK
        bytes map
        datetime createdAt
    }

    CoverageSourceMap {
        string hash PK
        bytes sourceMap
    }

    CoverageSnapshot {
        int id PK
        string provider
        string repoID
        string subject
        string subjectID
        string title
        datetime freezeTime
        string status
        bytes artifactZip
        string artifactHash
        string buildTarget
        json scene
    }

    CoverageClientPayloadIdempotency {
        string payloadHash PK
        datetime createdAt
    }
```

### Relations

| Relation | Cardinality | On delete | Notes |
| --- | --- | --- | --- |
| `CoverageBuild` → `Coverage` | 1:N | Cascade | PK=`buildHash`; Coverage.id = `buildHash\|sceneKey` |
| `CoverageBuild` → `CoverageMapRelation` | 1:N | Cascade | MapRelation shared per build |
| `Coverage` → `CoverageHit` | 1:N | Cascade | Composite FK `[buildHash, sceneKey]` |
| `Coverage` → `CoverageLock` | 1:0..1 | Cascade | Lock by `coverageID` |
| `CoverageMapRelation` → `CoverageMap` | N:1 | — | `coverageMapKey` = `coverageMapHash\|fileContentHash` |
| `CoverageMapRelation` → `CoverageSourceMap` | N:0..1 | — | `sourceMapHash` is null when missing |

### sceneKey

`sceneKey = hash(source + type + env + trigger …)`:

- **source**: `automation` / `manual` / `replay`
- **type**: `e2e` / `unit` / `integration`
- **env**: `test` / `staging` / `prod`
- **trigger**: `pipeline` / `schedule` / `manual`

### Standalone models

| Model | Table | Notes |
| --- | --- | --- |
| `CoverageSnapshot` | `canyon_next_coverage_snapshot` | Frozen snapshot/artifacts; `artifactHash` is unrelated to `CoverageBuild.buildHash`; logical link via `provider + repoID` |
| `CoverageClientPayloadIdempotency` | `canyon_next_coverage_client_payload_idempotency` | Idempotency for `POST /api/coverage/client` payload hashes |

## Coverage write path

```mermaid
sequenceDiagram
    participant Client
    participant API as "coverage client API"
    participant Idem as CoverageClientPayloadIdempotency
    participant Build as CoverageBuild
    participant Cov as Coverage
    participant Hit as CoverageHit
    participant Rel as CoverageMapRelation
    participant Map as CoverageMap
    participant SM as CoverageSourceMap

    Client->>API: Report coverage
    API->>Idem: Check payloadHash
    alt Exists
        Idem-->>API: Idempotent hit, skip
    else New
        API->>Build: upsert by buildHash
        API->>Map: reuse or write by hash
        API->>SM: optional sourceMap write
        API->>Rel: bind build and map
        API->>Cov: upsert by buildHash + sceneKey
        API->>Hit: upsert file hits
    end
```
